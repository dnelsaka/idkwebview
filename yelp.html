⚠️ Vulnerability PoC

This page demonstrates unauthorized access to the Android JS Bridge via the ActivityNotificationWebViewUrlCatcher injection.

1. Clipboard Poisoning

This button triggers Android.copyToClipboard() to replace the user's clipboard with an attacker-controlled string.

2. Arbitrary File Write (Public Pictures)

This triggers the openBottomSharingScreen interface to decode a base64 string and write a file to the device's public directory.
